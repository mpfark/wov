

# BHP System — Updated Success Floor

One small tweak to the training formula: the minimum success chance at rank 6+ changes from 10% to 1%.

```text
Training cost:      20 BHP per attempt
Success chance:     max(1%, 95% - rank × 15%)

Rank 0 → 1:  95%
Rank 1 → 2:  80%
Rank 2 → 3:  65%
Rank 3 → 4:  50%
Rank 4 → 5:  35%
Rank 5 → 6:  20%
Rank 6 → 7:   5%
Rank 7+:      1% (floor)
```

This makes deep stacking into a single stat extremely punishing while still technically possible — a true gamble for dedicated boss hunters.

Everything else from the previously approved plan remains the same. Ready to implement the full BHP system with this updated floor.
